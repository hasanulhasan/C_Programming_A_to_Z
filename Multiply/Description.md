# Multiply

You will be given two integers A and B. You need to give output their multiplication.

##### Input Format

Input will contain A and B
```bash
Constraints

-10^9 <= A,B <= 10^9
```

##### Output Format

- Output the multiplication

```bash
Sample Input 1

10 50

Sample Output 1

500

Sample Input 2

10000000 10000000

Sample Output 2

100000000000000

Sample Input 3

-100 62

Sample Output 3

-6200
